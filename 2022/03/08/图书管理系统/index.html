一 JavaWeb 项目环境搭建

1.1 开发环境和工具

操作系统: win10

内存: 4G以上

JDK: 8.0

开发工具: Intellij IDEA2021.3.2

服务器: Tomcat8.0

数据库工具: mysql5.7.37 + Navicat 15 for Mysql

1.2 项目技术

基于B/S结构

前端: HTML + CSS + JS + JQueryZ

后端: Servlet + JSP + MysqlV

1.3 数据库的创建

数据库名: bookdb

字符集: utf8

排序规则: utf8_general_ci 不区分大小写
