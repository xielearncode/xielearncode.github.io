一 Vi 和Vim编辑器

Vi : Linux的内置文本编辑器, 相当于是Windows的记事本

Vim : Vi 的增强版本

1.1 常用的三种模式

  • 正常模式: 直接进入时,
  • 插入模式: 按下i, I, o, O , a, A, r, R 进入插入模式
  • 命令行模式:

案例: 用vim 编辑一个Java文件

  1. 建立文件, 直接进入了正常模式

  1. 按下 i 进入插入模式

  1. 按下ESC 后输入 : 进入命令行模式 wq表示写入并退出

1.2 vi 和 vim 快捷键

  1. 拷贝当前行: yy 拷贝当前行向下五行: 5yy <======正常模式
  2. 删除当前行: dd 删除当前行向下五行 5dd <=======正常模式
  3. 查找: /关键字 n—>下一个 <========正常模式—–在进入—->命令行模式
  4. 设置文件行号: set nu 取消行号: set nonu <========命令行模式
  5. 命令行 : wq, :q, :q! <========命令行模式
  6. 大写G末尾行, 小写gg首行 <========一般模式
  7. 撤销 : 输入u <========一般模式
  8. 定位某一行: 数字(行号) + shift + g <========一般模式
